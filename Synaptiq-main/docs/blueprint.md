# **App Name**: UniLife AI

## Core Features:

- Smart Schedule Generator: Generates daily and weekly study schedules tailored to subjects, exam dates, syllabus size, difficulty, and available time.
- Adaptive Schedule Adjustment: Adapts schedules based on missed sessions, performance, fatigue, and overall health data to maintain an optimal study plan.
- Performance Analytics Dashboard: Tracks subject-wise progress, time spent, completion rate, and improvement trends, generating productivity and study efficiency scores with insights.
- AI-Powered Skill Matcher: Analyzes interests, strengths, academic background, and skills to recommend career paths, required skills, gap analysis, and learning roadmaps, leveraging AI as a tool.
- Integrated Learning Hub: Provides personalized learning resources (videos, courses, articles) based on study goals, tracked progress, and performance analytics.
- AI Study Suggestions: Leverages AI as a tool that analyzes performance, focus, deadlines, and health data to offer personalized study improvements, revision alerts, and productivity tips.
- Centralized User Profile: Provides secure storage for medical history, prescriptions, health metrics, study data and habits

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) to evoke focus and intelligence.
- Background color: Very light indigo (#E8EAF6).
- Accent color: Electric violet (#8E24AA) to add vibrancy and highlight key elements.
- Body text font: 'Inter', a sans-serif font, known for being legible at a variety of sizes.
- Headline text font: 'Space Grotesk', a sans-serif font with a techy look. It is paired with 'Inter'.
- Use clean, minimalist icons with a modern design. Icons should be consistent throughout the application to provide a seamless user experience.
- Employs a modern, responsive layout that adapts seamlessly to different screen sizes. Use of whitespace is maximized for readability.
- Subtle animations for transitions and user interactions to enhance the user experience and provide visual feedback.
